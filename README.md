# Project Overview
Designed and developed an end-to-end Retail Sales Analytics solution using SAP Datasphere, SAP HANA Cloud, and SAP Analytics Cloud (SAC). Created dimensional data models for Customer, Product, Store, and Sales entities. Built Graphical Views and Analytical Models to establish a semantic layer for reporting and analytics. Developed interactive SAC dashboards with KPIs, filters, drill-down capabilities, and visualizations for Executive, Product, Store, Customer, and Sales Performance analysis. Demonstrated expertise in data modeling, analytical modeling, dashboard development, and SAP analytics technologies.
